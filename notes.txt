no - 4

const nums = [1, 2, 3]
Math.min(nums)    // NaN
Math.max(nums)    // Nan
That is because Math.min or Math.max functions expect distinct variables and not an array. 

var nums = [1, 2, 3]
Math.min.apply(Math, nums)    // 1
Math.max.apply(Math, nums)    // 3
Math.min.apply(null, nums)    // 1
Math.max.apply(null, nums)    // 3
